使用
----
默认快捷键 ssh, 查看已添加服务器列表。

添加服务器记录格式: 
    <nickname> <host> <user> <password> [<root-password>] | do
    昵称 地址 登录用户名 登录密码 [root账户密码]

功能
----

1. 按`回车`打开终端并进入自动登录服务器
2. 按`Shift+回车`删除服务器记录
3. 按`Command+回车`复制服务器记录
